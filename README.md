# JAWOR 2023 Company Website

A professional, fully responsive Next.js website for JAWOR 2023 General Supplies and Construction Company Limited.

## Features

- **Modern Tech Stack**: Built with Next.js 14, TypeScript, and Tailwind CSS
- **Fully Responsive**: Mobile-first design that works on all devices
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Smooth Animations**: Fade-in and slide-up effects for better UX
- **Sticky Navigation**: Easy navigation with dropdown menus
- **Contact Form**: Functional contact form with validation
- **Interactive Map**: Google Maps integration for location
- **Career Portal**: Separate pages for jobs and internships

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Clone or download the project
2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Run the development server:

```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```
